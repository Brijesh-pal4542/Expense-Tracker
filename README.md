# Expense-Tracker
A modern and responsive Expense Tracker built with React that helps users manage daily expenses, monitor spending habits, and track budgets efficiently. The application features a clean dashboard, expense management system, and persistent data storage using Local Storage.

# Features
- Add and delete expenses
- Categorize expenses for better tracking
- Dashboard with spending overview
- Budget monitoring and utilization tracking
- Persistent storage using Local Storage
- Responsive and modern UI
- Dark/Light mode support
- Sidebar navigation for seamless user experience

# Technologies Used
- React. + Vite
- React Router DOM
- JavaScript
- HTML5
- CSS3
- React Icons
- Local Storage support
